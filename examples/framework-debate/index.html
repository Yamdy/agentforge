// debate highlights

辩论焦点

核心攻击点

AgentForge 零社区、零 npm 发布，是概念验证不是生产框架。
DeepAgents 是 LangChain 的薄封装，427 行条件组装逻辑。
Mastra 5700 行上帝类，alpha 版本，Node 22 硬绑定。

核心防守点

自研框架 = 零供应链风险，完全主权，这是护城河不是劣势。
继承 LangGraph 生产级能力，省下精力做 85 项评测。
8700+ commits / 392 人，alpha 是迭代速度的体现。
