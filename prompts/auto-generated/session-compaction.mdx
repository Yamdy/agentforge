# SessionCompaction

## Description

Session message compaction utilities. Reduces message count by summarizing middle conversations while preserving context at boundaries.

## Signature

```typescript
function compactSession(session: Session, options?: CompactionOptions): Promise<CompactionResult>
function applyCompaction(session: Session, compactedMessages: SessionMessage[]): Session
```

## Types

### CompactionOptions

```typescript
interface CompactionOptions {
  maxMessages?: number      // Default: 50
  keepFirst?: number        // Default: 2
  keepLast?: number         // Default: 10
  summaryPrompt?: string    // Custom summary prompt
}
```

### CompactionResult

```typescript
interface CompactionResult {
  originalCount: number
  compactedCount: number
  summary: string
}
```

## Functions

### compactSession(session, options)

Summarizes middle messages, keeps first N and last N messages.

**Algorithm:**
1. If `session.messages.length <= maxMessages`, no compaction needed
2. Separate system messages (always kept)
3. Keep first `keepFirst` non-system messages
4. Keep last `keepLast` non-system messages
5. Summarize middle messages into system message
6. Return result with summary

**Returns:** `Promise<CompactionResult>`

### applyCompaction(session, compactedMessages)

Apply compacted messages to session, sets `compactedAt` timestamp.

**Returns:** `Session` - Updated session with new messages and timestamp

## Examples

### Example 1: Basic Compaction

```typescript
import { compactSession } from 'primo-agent';

const session = {
  id: 's1',
  title: 'Chat',
  messages: [
    { role: 'system', content: 'You are helpful.' },
    ...Array(30).fill(null).map((_, i) => ({
      role: i % 2 === 0 ? 'user' : 'assistant',
      content: `Message ${i}`
    })),
  ],
  createdAt: Date.now(),
  updatedAt: Date.now(),
};

const result = await compactSession(session, {
  maxMessages: 20,
  keepFirst: 2,
  keepLast: 10,
});

console.log(result.originalCount);    // 31
console.log(result.compactedCount);   // 14 (2 first + 1 summary + 10 last + 1 system)
console.log(result.summary);         // "Session had 18 intermediate messages..."
```

### Example 2: Apply and Update

```typescript
import { compactSession, applyCompaction } from 'primo-agent';

const result = await compactSession(session);
const compacted = [
  session.messages[0],  // system
  ...session.messages.slice(0, 2),  // first 2
  { role: 'system', content: `[Summary: ${result.summary}]` },
  ...session.messages.slice(-15),  // last 15
];

const updated = applyCompaction(session, compacted);
console.log(updated.compactedAt);  // Timestamp set
```
