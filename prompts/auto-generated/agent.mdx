# Agent

## Description

Core AI Agent engine that orchestrates LLM interactions, tool execution, and plugin lifecycle. Supports both synchronous and streaming execution modes with comprehensive event handling.

## Signature

```typescript
class Agent {
  constructor(adapter: LLMAdapter, history: HistoryManager, registry?: ToolRegistry, options?: AgentOptions)
  run(userInput: string, options?: RunOptions): Promise<string>
  runStream(userInput: string, options?: RunOptions): AsyncGenerator<StreamEvent>
  getState(): TaskState
  cancel(): void
  registerPlugin(plugin: Plugin): void
  getPluginManager(): PluginManager
}
```

## Constructor Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| adapter | LLMAdapter | Yes | LLM adapter for chat completion |
| history | HistoryManager | Yes | Message history manager |
| registry | ToolRegistry | No | Tool registry (defaults to new instance) |
| options.maxSteps | number | No | Maximum steps (default: Infinity) |
| options.pluginManager | PluginManager | No | Plugin manager instance |
| options.logger | Logger | No | Custom logger |

## Methods

### run(userInput, options)

Execute agent synchronously and return full response as string.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| userInput | string | Yes | User input prompt |
| options.sessionMessages | Message[] | No | Prior session messages |
| options.onText | (text: string) => void | No | Text delta callback |
| options.onToolCallStart | (id: string, name: string) => void | No | Tool call start callback |
| options.onToolCallEnd | (id: string, result?: string) => void | No | Tool call end callback |
| options.onStep | (step: number, maxSteps: number) => void | No | Step progress callback |
| options.onStateChange | (state: TaskState) => void | No | State change callback |
| options.onError | (error: Error) => void | No | Error callback |

**Returns:** `Promise<string>` - Complete response

### runStream(userInput, options)

Execute agent with streaming events.

**Returns:** `AsyncGenerator<StreamEvent>` - Stream of events

**Event types:**
- `{ type: 'text', content: string }` - Text delta
- `{ type: 'tool_call_start', id: string, name: string }` - Tool call start
- `{ type: 'tool_call_delta', id: string, arguments: string }` - Tool arguments
- `{ type: 'tool_call_end', id: string, result?: string }` - Tool result
- `{ type: 'done', response: LLMResponse }` - Completion

### getState()

Returns current agent task state.

**Returns:** `TaskState` - `{ status, step, maxSteps, error? }`

### cancel()

Cancel current agent execution. Transitions state to 'cancelled'.

## Examples

### Example 1: Basic Synchronous Run

```typescript
import { Agent, AIAdapter, InMemoryHistory, ToolRegistry } from 'primo-agent';

const adapter = new AIAdapter({ model: 'gpt-4', apiKey: process.env.OPENAI_KEY });
const history = new InMemoryHistory();
const registry = new ToolRegistry();
registry.register(calculatorTool);
registry.register(searchTool);

const agent = new Agent(adapter, history, registry, { maxSteps: 10 });

const response = await agent.run('What is 123 * 456?');
console.log(response);
```

### Example 2: Streaming with Callbacks

```typescript
const stream = agent.runStream('Search for AI news', {
  onText: (text) => process.stdout.write(text),
  onToolCallStart: (id, name) => console.log(`\n[Tool: ${name}]`),
  onToolCallEnd: (id, result) => console.log(`Result: ${result}`),
  onStep: (step, max) => console.log(`Step ${step}/${max}`),
});

for await (const event of stream) {
  // Handle events
}
```

### Example 3: Plugin Registration

```typescript
const myPlugin = {
  name: 'my-plugin',
  version: '1.0.0',
  hooks: {
    'agent.step': async (input, output) => {
      console.log(`Step ${input.step} started`);
    },
    'tool.execute.after': async (input, output) => {
      console.log(`Tool ${input.tool} returned: ${input.result}`);
    },
  },
};

agent.registerPlugin(myPlugin);
```

## Plugin Hooks

Available hooks triggered during agent lifecycle:

| Hook | Input | Description |
|------|-------|-------------|
| agent.start | `{ userInput: string }` | Before run starts |
| agent.step | `{ step: number, maxSteps: number }` | Each iteration |
| tool.execute.before | `{ tool: string, args: object }` | Before tool call |
| tool.execute.after | `{ tool: string, args: object, result: string }` | After tool call |
| agent.error | `{ error: string }` | On error |
| agent.complete | `{ userInput: string, response: string }` | On completion |
| state.change | `{ from: TaskStatus, to: TaskStatus }` | State transitions |
