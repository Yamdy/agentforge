# Logger

## Description

Structured logging service with log level filtering, child loggers, and observable streams for log consumption. Uses singleton pattern for global access.

## Signature

```typescript
class LogService implements Logger {
  constructor(service?: string, logSubject?: Subject<LogEntry>)
  static getInstance(): LogService
  static setLogSubject(subject: Subject<LogEntry>): void
  setLevel(level: LogLevel): void
  debug(message: string, meta?: Record<string, unknown>): void
  info(message: string, meta?: Record<string, unknown>): void
  warn(message: string, meta?: Record<string, unknown>): void
  error(message: string, meta?: Record<string, unknown>): void
  child(service: string): Logger
  observable(): Observable<LogEntry>
  observableByLevel(level: LogLevel): Observable<LogEntry>
  observableByService(service: string): Observable<LogEntry>
}

function createLogger(service: string): Logger
```

## Types

### LogLevel

```typescript
type LogLevel = 'debug' | 'info' | 'warn' | 'error'
```

### LogEntry

```typescript
interface LogEntry {
  timestamp: Date;
  level: LogLevel;
  service: string;
  message: string;
  meta?: Record<string, unknown>;
  traceId?: string;
}
```

### Logger Interface

```typescript
interface Logger {
  debug(message: string, meta?: Record<string, unknown>): void
  info(message: string, meta?: Record<string, unknown>): void
  warn(message: string, meta?: Record<string, unknown>): void
  error(message: string, meta?: Record<string, unknown>): void
  child(service: string): Logger
}
```

## Methods

### setLevel(level)

Set minimum log level. Below this level, logs are suppressed.

| Level | Numeric | Output |
|-------|---------|--------|
| debug | 0 | All logs |
| info | 1 | info, warn, error |
| warn | 2 | warn, error |
| error | 3 | error only |

### child(service)

Create child logger with new service name.

**Returns:** `Logger` - New LogService instance

### observable(), observableByLevel(), observableByService()

Get RxJS observables for log consumption.

## Examples

### Example 1: Basic Logging

```typescript
import { createLogger } from 'primo-agent';

const log = createLogger('my-service');

log.info('Application started');
log.warn('Memory usage high', { usage: '85%' });
log.error('Connection failed', { host: 'db.example.com', code: 'ECONNREFUSED' });
```

### Example 2: Level Filtering

```typescript
log.setLevel('warn');  // Only warnings and errors

log.info('This will be suppressed');
log.warn('This will show');  // Shows
```

### Example 3: Observable Stream

```typescript
import { filter } from 'rxjs';

log.observableByLevel('error').subscribe((entry) => {
  console.error(`[${entry.timestamp}] ${entry.message}`, entry.meta);
});

// Filter by service
log.observableByService('agent').subscribe((entry) => {
  console.log(`[agent] ${entry.message}`);
});
```

### Example 4: Child Loggers

```typescript
const agentLog = log.child('agent');
const toolLog = log.child('tool');

agentLog.info('Agent started');
toolLog.info('Tool executed', { tool: 'calculator' });
// Output: [agent] Agent started
// Output: [tool] Tool executed
```
