# Storage

## Description

通用文件系统存储模块，基于 JSON 文件存储，支持读写锁保证并发安全。参考 opencode 实现。

## Signature

```typescript
namespace Storage {
  async function read<T>(key: string[]): Promise<T>
  async function write<T>(key: string[], content: T): Promise<void>
  async function update<T>(key: string[], fn: (draft: T) => void): Promise<T>
  async function remove(key: string[]): Promise<void>
  async function list(prefix: string[]): Promise<string[][]>
}
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| key | string[] | Yes | 存储键路径，如 `['session', 'id']` |
| content | T | Yes (write) | 要写入的内容 |
| fn | function | Yes (update) | 修改数据的回调函数 |
| prefix | string[] | Yes (list) | 列表前缀 |

## Returns

| Function | Returns | Description |
|----------|---------|-------------|
| read | Promise\<T\> | 读取的数据，不存在则抛出 NotFoundError |
| write | Promise\<void\> | 写入成功 |
| update | Promise\<T\> | 更新后的数据 |
| remove | Promise\<void\> | 删除成功 |
| list | Promise\<string[][]\> | 所有匹配的键路径 |

## Examples

```typescript
import { Storage, NotFoundError } from 'primo-agent';

// 写入数据
await Storage.write(['session', 'abc123'], {
  id: 'abc123',
  title: 'My Chat',
  messages: [],
  createdAt: Date.now(),
  updatedAt: Date.now(),
});

// 读取数据
try {
  const session = await Storage.read<Session>(['session', 'abc123']);
} catch (e) {
  if (e instanceof NotFoundError) {
    console.log('Session not found');
  }
}

// 更新数据
await Storage.update(['session', 'abc123'], (draft) => {
  draft.title = 'Updated Title';
  draft.updatedAt = Date.now();
});

// 删除数据
await Storage.remove(['session', 'abc123']);

// 列出所有 session
const sessions = await Storage.list(['session']);
```

## NotFoundError

当读取的文件不存在时抛出。

```typescript
import { NotFoundError } from 'primo-agent';

try {
  await Storage.read(['session', 'missing']);
} catch (e) {
  if (e instanceof NotFoundError) {
    console.log(e.message); // "Resource not found: /path/to/file.json"
  }
}
```
