# Types

## Description

Central type definitions and Zod schemas for the agent system. Includes message, tool, LLM response, stream events, and task state machine types.

## Schemas

### MessageSchema

```typescript
const MessageSchema = z.object({
  role: z.enum(['system', 'user', 'assistant']),
  content: z.string(),
})
type Message = z.infer<typeof MessageSchema>
```

### ToolSchema

```typescript
const ToolSchema = z.object({
  name: z.string().min(1),
  description: z.string(),
  parameters: ToolParametersSchema.optional(),
  execute: z.custom<(args) => Promise<string>>(),
})
type Tool = z.infer<typeof ToolSchema>
```

### ToolParametersSchema

```typescript
const ToolParametersSchema = z.object({
  type: z.literal('object'),
  properties: z.record(z.string(), z.any()),
  required: z.array(z.string()).optional(),
})
```

### StreamEventSchema

Discriminated union of event types:
- `{ type: 'text', content: string }`
- `{ type: 'tool_call_start', id: string, name: string }`
- `{ type: 'tool_call_delta', id: string, arguments: string }`
- `{ type: 'tool_call_end', id: string, result?: string }`
- `{ type: 'done', response: LLMResponse }`

### LLMResponseSchema

```typescript
const LLMResponseSchema = z.object({
  content: z.string().nullable(),
  toolCalls: z.array(ToolCallSchema).optional(),
  finishReason: z.enum(['stop', 'tool-calls', 'length', 'error']).optional(),
})
```

## Examples

### Example 1: Using Schemas

```typescript
import { MessageSchema, ToolSchema, validateTool } from 'primo-agent';

const msg = MessageSchema.parse({
  role: 'user',
  content: 'Hello'
});

const tool = validateTool({
  name: 'my-tool',
  description: 'Does something',
  execute: async (args) => 'result'
});
```

### Example 2: Implementing HistoryManager

```typescript
import { HistoryManager, Message } from 'primo-agent';

class MyHistory implements HistoryManager {
  private messages: Message[] = [];
  
  add(role, content) {
    this.messages.push({ role, content });
  }
  
  addToolResult(id, name, result) {
    this.messages.push({
      role: 'user',
      content: `[TOOL] ${name}: ${result}`
    });
  }
  
  getMessages() {
    return [...this.messages];
  }
  
  clear() {
    this.messages = [];
  }
}
```
