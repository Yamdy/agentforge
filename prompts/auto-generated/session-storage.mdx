# Session Storage

## Description

SQLite-based session persistence layer using sql.js (WebAssembly). Provides async CRUD operations for conversation sessions with message history.

## Signature

```typescript
// Initialization
function initSessionStorage(): Promise<SqlJsDatabase>
function getDatabase(): Promise<SqlJsDatabase>
function closeSessionStorage(): void

// Session CRUD
function createSession(id: string, title: string, options?: SessionOptions): Promise<Session>
function getSession(id: string): Promise<Session | null>
function listSessions(options?: ListOptions): Promise<Session[]>
function updateSession(id: string, updates: Partial<Session>): Promise<Session | null>
function addMessageToSession(id: string, message: SessionMessage): Promise<Session | null>
function deleteSession(id: string): Promise<boolean>
function markSessionCompacted(id: string): Promise<void>
```

## Types

### Session

```typescript
interface Session {
  id: string;
  title: string;
  messages: SessionMessage[];
  createdAt: number;      // Unix timestamp
  updatedAt: number;      // Unix timestamp
  compactedAt?: number;   // Unix timestamp
  parentId?: string;      // For session branching
  projectId?: string;     // For organization
}
```

### SessionMessage

```typescript
interface SessionMessage {
  role: 'system' | 'user' | 'assistant';
  content: string;
  timestamp?: number;
}
```

## Functions

### initSessionStorage()

Initialize SQLite database. Creates `data/sessions.db` if not exists, with indexes on `updated_at`, `parent_id`.

**Returns:** `Promise<SqlJsDatabase>`

### createSession(id, title, options)

Create new session with optional parent/project and initial messages.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | string | Yes | Unique session ID |
| title | string | Yes | Session title |
| options.parentId | string | No | Parent session ID |
| options.projectId | string | No | Project ID |
| options.messages | SessionMessage[] | No | Initial messages |

**Returns:** `Promise<Session>`

### getSession(id)

Retrieve session by ID.

**Returns:** `Promise<Session | null>` - Null if not found

### listSessions(options)

List sessions with pagination and filtering.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| options.limit | number | No | Max results (default: 50) |
| options.offset | number | No | Pagination offset |
| options.parentId | string | No | Filter by parent |
| options.projectId | string | No | Filter by project |

**Returns:** `Promise<Session[]>` - Ordered by `updated_at DESC`

### updateSession(id, updates)

Update session fields.

**Returns:** `Promise<Session | null>`

### addMessageToSession(id, message)

Append message to session. Adds timestamp if not provided.

**Returns:** `Promise<Session | null>`

### deleteSession(id)

Delete session by ID.

**Returns:** `Promise<boolean>` - True if deleted

## Database Schema

```sql
CREATE TABLE sessions (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  messages TEXT NOT NULL DEFAULT '[]',
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL,
  compacted_at INTEGER,
  parent_id TEXT,
  project_id TEXT
)

CREATE INDEX idx_sessions_updated_at ON sessions(updated_at DESC)
CREATE INDEX idx_sessions_parent_id ON sessions(parent_id)
```

## Examples

### Example 1: Basic Session Operations

```typescript
import { initSessionStorage, createSession, getSession, addMessageToSession } from 'primo-agent';

await initSessionStorage();

// Create session
const session = await createSession(
  'session_123',
  'My Chat',
  { projectId: 'project_abc' }
);

// Add messages
await addMessageToSession('session_123', {
  role: 'user',
  content: 'Hello AI'
});

await addMessageToSession('session_123', {
  role: 'assistant',
  content: 'Hello! How can I help?'
});

// Retrieve
const updated = await getSession('session_123');
console.log(updated.messages.length); // 2
```

### Example 2: List with Filters

```typescript
const sessions = await listSessions({
  limit: 10,
  offset: 0,
  projectId: 'project_abc'
});

for (const s of sessions) {
  console.log(`${s.title} (${s.messages.length} messages)`);
}
```

### Example 3: Session Branching

```typescript
// Create child session from parent
const parent = await createSession('parent_1', 'Parent Session');

const child = await createSession(
  'child_1',
  'Child Branch',
  { parentId: parent.id, projectId: parent.projectId }
);

// List children of parent
const children = await listSessions({ parentId: 'parent_1' });
```
