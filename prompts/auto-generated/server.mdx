# Server

## Description

HTTP server and API endpoints for running AI agents. Built on Hono framework with SSE streaming support, session management, and automatic message compaction.

## Signature

```typescript
function createApp(config: ServerConfig & { agent?: AgentRunner }): Hono
async function startServer(config: ServerConfig & { agent?: AgentRunner }): Promise<Server>
```

## Configuration

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| port | number | No | Server port (default: 3000) |
| apiKey | string | No | API key for authentication |
| corsOrigins | string[] | No | Allowed CORS origins (default: "*") |
| compactionThreshold | number | No | Message count threshold for compaction (default: 20) |
| compactionEnabled | boolean | No | Enable session compaction (default: true) |
| agent | AgentRunner | No | Agent instance to attach |

## API Endpoints

### Agent Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/agent/run` | Run agent synchronously |
| POST | `/api/agent/run/stream` | Run agent with SSE streaming |
| GET | `/api/agent/status` | Get current agent state |

### Session Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/sessions` | Create new session |
| GET | `/api/sessions` | List sessions (supports limit, offset, parentId, projectId) |
| GET | `/api/sessions/:id` | Get session by ID |
| PUT | `/api/sessions/:id` | Update session |
| DELETE | `/api/sessions/:id` | Delete session |
| POST | `/api/sessions/:sessionID/run` | Run agent in session context |
| POST | `/api/sessions/:sessionID/run/stream` | Stream agent in session context |

### System Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/health` | Health check |
| GET | `/openapi.json` | OpenAPI specification |

## Request/Response Formats

### POST /api/agent/run

**Request:**
```json
{
  "input": "Your prompt here",
  "sessionId": "optional-session-id"
}
```

**Response:**
```json
{
  "result": "Agent response"
}
```

### POST /api/agent/run/stream

**Response:** Server-Sent Events (SSE)

```typescript
// Event types
{ type: 'text', content: string }
{ type: 'tool_call_start', id: string, name: string }
{ type: 'tool_call_delta', id: string, arguments: string }
{ type: 'tool_call_end', id: string, result: string }
{ type: 'done' }
{ type: 'error', error: string }
```

## Session Management

Sessions store conversation history with automatic compaction:

1. When messages exceed `compactionThreshold` (default 20)
2. Keeps first 2 and last 15 messages
3. Replaces middle with summary system message
4. Updates session in database

## Examples

### Example 1: Create and Start Server

```typescript
import { createApp, startServer } from 'primo-agent';
import { Agent, AIAdapter, InMemoryHistory, ToolRegistry } from 'primo-agent';

const adapter = new AIAdapter({ model: 'gpt-4', apiKey: process.env.OPENAI_KEY });
const history = new InMemoryHistory();
const registry = new ToolRegistry();
// ... register tools

const agent = new Agent(adapter, history, registry);

const app = createApp({
  agent,
  apiKey: process.env.API_KEY,
  corsOrigins: ['https://yourapp.com'],
  compactionThreshold: 30,
});

await startServer({ port: 3000, agent });
```

### Example 2: SSE Streaming Client

```typescript
const response = await fetch('http://localhost:3000/api/agent/run/stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ input: 'Hello AI' }),
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  
  const lines = decoder.decode(value).split('\n');
  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const event = JSON.parse(line.slice(6));
      console.log('Event:', event.type, event.content || '');
    }
  }
}
```

### Example 3: Session Persistence

```typescript
// Create session
const session = await fetch('http://localhost:3000/api/sessions', {
  method: 'POST',
  body: JSON.stringify({ title: 'My Chat', messages: [] }),
}).then(r => r.json());

// Continue conversation
const result = await fetch(`http://localhost:3000/api/sessions/${session.id}/run/stream`, {
  method: 'POST',
  body: JSON.stringify({ input: 'What is AI?' }),
});
```
