# SessionAPI

## Description

Facade API for session management. Wraps storage layer with async interface, provides typed methods for session CRUD operations.

## Signature

```typescript
interface SessionAPI {
  init(): Promise<void>
  close(): void
  create(options?: SessionCreateOptions): Promise<Session>
  get(id: string): Promise<Session | null>
  list(options?: SessionListOptions): Promise<Session[]>
  update(id: string, updates: Partial<Session>): Promise<Session | null>
  addMessage(id: string, message: SessionMessage): Promise<Session | null>
  delete(id: string): Promise<boolean>
}

function createSessionAPI(): SessionAPI
```

## Types

### SessionCreateOptions

```typescript
interface SessionCreateOptions {
  title?: string
  messages?: SessionMessage[]
  parentId?: string
  projectId?: string
}
```

### SessionListOptions

```typescript
interface SessionListOptions {
  limit?: number
  offset?: number
  parentId?: string
  projectId?: string
}
```

## Methods

### init()

Initialize storage (calls `initSessionStorage`). Must be called before operations.

### close()

Close storage connection.

### create(options)

Create new session with auto-generated ID.

**Returns:** `Promise<Session>`

### get(id)

Get session by ID.

**Returns:** `Promise<Session | null>`

### list(options)

List sessions with pagination.

**Returns:** `Promise<Session[]>`

### update(id, updates)

Update session fields.

**Returns:** `Promise<Session | null>`

### addMessage(id, message)

Append message to session.

**Returns:** `Promise<Session | null>`

### delete(id)

Delete session.

**Returns:** `Promise<boolean>`

## Examples

### Example 1: Basic Usage

```typescript
import { createSessionAPI } from 'primo-agent';

const api = createSessionAPI();
await api.init();

// Create session
const session = await api.create({ title: 'My Chat' });

// Add messages
await api.addMessage(session.id, { role: 'user', content: 'Hello' });
await api.addMessage(session.id, { role: 'assistant', content: 'Hi there!' });

// Get updated session
const updated = await api.get(session.id);
console.log(updated.messages.length); // 2
```

### Example 2: With Parent/Project

```typescript
const parent = await api.create({ title: 'Parent' });
const child = await api.create({
  title: 'Branch',
  parentId: parent.id,
  projectId: 'project-1'
});

const children = await api.list({ parentId: parent.id });
```
