# MCP Integration

## Description

Model Context Protocol (MCP) integration for connecting to external MCP servers. Supports both local (stdio) and remote (HTTP/SSE) servers, with automatic tool discovery and toolkit management.

## Architecture

```
┌─────────────────────────────────────────┐
│         Primo Agent                      │
│  ┌─────────────────────────────────┐   │
│  │      MCP Client                │   │
│  │  ┌─────────────────────────┐   │   │
│  │  │  Local (stdio)          │   │   │
│  │  └─────────────────────────┘   │   │
│  │  ┌─────────────────────────┐   │   │
│  │  │  Remote (HTTP/SSE)     │   │   │
│  │  └─────────────────────────┘   │   │
│  └──────────────┬──────────────────┘   │
│                 │                        │
│  ┌──────────────▼──────────────────┐   │
│  │      MCP Toolkit                │   │
│  │  - Tool Groups                  │   │
│  │  - Activation/Deactivation      │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
             │
             ▼
    ┌─────────────────┐
    │  MCP Servers    │
    │  (Local/Remote) │
    └─────────────────┘
```

## Signature

```typescript
const MCP: {
  client: {
    init(): Promise<void>;
    add(name: string, serverConfig: McpServerConfig): Promise<McpStatus>;
    remove(name: string): Promise<void>;
    connect(name: string, cfg?: McpServerConfig): Promise<McpStatus>;
    disconnect(name: string): Promise<void>;
    status(): Record<string, McpStatus>;
    tools(): Promise<Record<string, Tool>>;
  };
  Toolkit: {
    refreshTools(): Promise<void>;
    registerGroup(name: string, tools: Tool[]): void;
    activateGroup(name: string): void;
    deactivateGroup(name: string): void;
    addToBasic(serverName: string): void;
    removeFromBasic(serverName: string): void;
    getTools(groups?: string[]): Tool[];
  };
  config: {
    load(): Promise<void>;
    getServers(): Record<string, McpServerConfig>;
    getServer(name: string): McpServerConfig | undefined;
    setServer(name: string, config: McpServerConfig): Promise<void>;
    removeServer(name: string): Promise<void>;
  };
  schemas: {
    McpLocalConfig: z.ZodSchema;
    McpRemoteConfig: z.ZodSchema;
    McpServerConfig: z.ZodSchema;
    McpStatus: z.ZodSchema;
  };
};
```

## MCP Client Methods

### MCP.client.init()

Initialize MCP client and connect to configured servers.

**Returns:** `Promise<void>`

### MCP.client.add(name, serverConfig)

Add a new MCP server.

| Parameter    | Type            | Required | Description          |
| ------------ | --------------- | -------- | -------------------- |
| name         | string          | Yes      | Server name          |
| serverConfig | McpServerConfig | Yes      | Server configuration |

**Returns:** `Promise<McpStatus>` - Connection status

### MCP.client.remove(name)

Remove an MCP server.

| Parameter | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| name      | string | Yes      | Server name |

**Returns:** `Promise<void>`

### MCP.client.connect(name, cfg?)

Connect to an MCP server.

| Parameter | Type            | Required | Description                                        |
| --------- | --------------- | -------- | -------------------------------------------------- |
| name      | string          | Yes      | Server name                                        |
| cfg       | McpServerConfig | No       | Server config (uses stored config if not provided) |

**Returns:** `Promise<McpStatus>` - Connection status

### MCP.client.disconnect(name)

Disconnect from an MCP server.

| Parameter | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| name      | string | Yes      | Server name |

**Returns:** `Promise<void>`

### MCP.client.status()

Get status of all MCP servers.

**Returns:** `Record<string, McpStatus>` - Server status map

### MCP.client.tools()

Get all tools from connected MCP servers.

**Returns:** `Promise<Record<string, Tool>>` - Tools from all servers

## MCP Toolkit Methods

### MCP.Toolkit.refreshTools()

Refresh tools from connected MCP servers.

**Returns:** `Promise<void>`

### MCP.Toolkit.registerGroup(name, tools)

Register a custom tool group.

| Parameter | Type   | Required | Description    |
| --------- | ------ | -------- | -------------- |
| name      | string | Yes      | Group name     |
| tools     | Tool[] | Yes      | Tools in group |

### MCP.Toolkit.activateGroup(name)

Activate a tool group.

| Parameter | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| name      | string | Yes      | Group name  |

### MCP.Toolkit.deactivateGroup(name)

Deactivate a tool group.

| Parameter | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| name      | string | Yes      | Group name  |

### MCP.Toolkit.addToBasic(serverName)

Add a server to the basic tool group.

| Parameter  | Type   | Required | Description |
| ---------- | ------ | -------- | ----------- |
| serverName | string | Yes      | Server name |

### MCP.Toolkit.removeFromBasic(serverName)

Remove a server from the basic tool group.

| Parameter  | Type   | Required | Description |
| ---------- | ------ | -------- | ----------- |
| serverName | string | Yes      | Server name |

### MCP.Toolkit.getTools(groups?)

Get tools from specified groups.

| Parameter | Type     | Required | Description                         |
| --------- | -------- | -------- | ----------------------------------- |
| groups    | string[] | No       | Group names (defaults to ['basic']) |

**Returns:** `Tool[]` - Active tools

## Configuration File

Create `mcp.config.json` in project root:

```json
{
  "servers": {
    "filesystem": {
      "type": "local",
      "command": ["npx", "-y", "@modelcontextprotocol/server-filesystem", "/path/to/dir"],
      "enabled": true
    },
    "brave-search": {
      "type": "remote",
      "url": "https://mcp.example.com/brave-search",
      "enabled": true
    }
  }
}
```

## Examples

### Example 1: Initialize MCP

```typescript
import { MCP } from 'primo-agent';

// Initialize MCP client (loads config and connects)
await MCP.client.init();

// Check server statuses
const statuses = MCP.client.status();
console.log('Server statuses:', statuses);
```

### Example 2: Add Local MCP Server

```typescript
import { MCP } from 'primo-agent';

const status = await MCP.client.add('filesystem', {
  type: 'local',
  command: ['npx', '-y', '@modelcontextprotocol/server-filesystem', '.'],
  enabled: true,
});

console.log('Connection status:', status);
```

### Example 3: Add Remote MCP Server

```typescript
import { MCP } from 'primo-agent';

const status = await MCP.client.add('brave-search', {
  type: 'remote',
  url: 'https://mcp.example.com/brave-search',
  headers: {
    Authorization: 'Bearer YOUR_TOKEN',
  },
  enabled: true,
});

console.log('Connection status:', status);
```

### Example 4: Get and Use MCP Tools

```typescript
import { MCP, Agent, AIAdapter, InMemoryHistory, ToolRegistry } from 'primo-agent';

await MCP.client.init();

// Refresh tools from MCP servers
await MCP.Toolkit.refreshTools();

// Get active tools
const mcpTools = MCP.Toolkit.getTools(['basic']);

// Create agent with MCP tools
const registry = new ToolRegistry();
for (const tool of mcpTools) {
  registry.register(tool);
}

const agent = new Agent(new AIAdapter({ model: 'gpt-4' }), new InMemoryHistory(), registry);

// Agent can now use MCP tools!
const response = await agent.run('List the files in the current directory');
```

### Example 5: Manage Tool Groups

```typescript
import { MCP } from 'primo-agent';

await MCP.client.init();

// Add specific servers to basic group
MCP.Toolkit.addToBasic('filesystem');
MCP.Toolkit.addToBasic('brave-search');

// Register custom tool group
const customTools = [
  // ... your custom tools
];
MCP.Toolkit.registerGroup('custom', customTools);

// Activate/deactivate groups
MCP.Toolkit.activateGroup('custom');
MCP.Toolkit.deactivateGroup('basic');

// Get tools from multiple groups
const tools = MCP.Toolkit.getTools(['basic', 'custom']);
```

### Example 6: Disconnect and Remove Servers

```typescript
import { MCP } from 'primo-agent';

// Disconnect a server
await MCP.client.disconnect('old-server');

// Remove a server completely
await MCP.client.remove('old-server');
```

### Example 7: Validate Configurations

```typescript
import { MCP } from 'primo-agent';

// Validate local config
const localConfig = {
  type: 'local' as const,
  command: ['echo', 'hello'],
};
const localResult = MCP.schemas.McpServerConfig.safeParse(localConfig);
console.log('Local config valid:', localResult.success);

// Validate remote config
const remoteConfig = {
  type: 'remote' as const,
  url: 'https://example.com/mcp',
};
const remoteResult = MCP.schemas.McpServerConfig.safeParse(remoteConfig);
console.log('Remote config valid:', remoteResult.success);
```
