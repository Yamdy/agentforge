# AIAdapter

## Description

LLM adapter implementing the `LLMAdapter` interface. Wraps AI SDK with OpenAI-compatible provider, supporting tool calling, streaming, and automatic schema conversion.

## Signature

```typescript
class AIAdapter implements LLMAdapter {
  constructor(config: AIAdapterConfig)
  setTools(tools: Tool[]): void
  chat(messages: Message[]): Promise<LLMResponse>
  chatStream(messages: Message[]): AsyncGenerator<StreamEvent>
}
```

## Constructor Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| config.model | string | Yes | Model identifier (e.g., 'gpt-4', 'claude-3') |
| config.apiKey | string | No | API key (default: process.env.OPENAI_API_KEY) |
| config.baseURL | string | No | Custom endpoint for compatible APIs |
| config.useTools | boolean | No | Enable tool calling (default: true) |

## Interface Implementation

### LLMAdapter

```typescript
interface LLMAdapter {
  chat(messages: Message[]): Promise<LLMResponse>
  chatStream(messages: Message[]): AsyncGenerator<StreamEvent>
}
```

## Methods

### setTools(tools)

Register tools for LLM to call. Tools must have `name`, `description`, `parameters`, and `execute`.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| tools | Tool[] | Yes | Array of tool definitions |

### chat(messages)

Synchronous chat completion. Returns full response when done.

**Returns:** `Promise<LLMResponse>` - `{ content, toolCalls?, finishReason }`

### chatStream(messages)

Streaming chat with event-based tool handling.

**Returns:** `AsyncGenerator<StreamEvent>`

**Event types:**
| Event | Fields | Description |
|-------|--------|-------------|
| text | `{ type: 'text', content: string }` | Text delta |
| tool_call_start | `{ type: 'tool_call_start', id: string, name: string }` | Tool call begins |
| tool_call_delta | `{ type: 'tool_call_delta', id: string, arguments: string }` | Tool arguments JSON |
| tool_call_end | `{ type: 'tool_call_end', id: string, result: string }` | Tool result |
| done | `{ type: 'done', response: LLMResponse }` | Completion |

## Tool Schema Conversion

Tools are converted from PrimoAgent format to AI SDK format:

```typescript
// Input (PrimoAgent)
{ name: 'calculator', description: '...', parameters: { type: 'object', properties: { expr: { type: 'string' } }, required: ['expr'] }, execute: async (args) => ... }

// Output (AI SDK)
{ calculator: { description: '...', inputSchema: z.object({ expr: z.string() }), execute: async (args) => ... } }
```

Supported parameter types: `string`, `number`, `boolean`. Other types default to `z.any()`.

## Examples

### Example 1: Basic Chat

```typescript
import { AIAdapter } from 'primo-agent';

const adapter = new AIAdapter({
  model: 'gpt-4',
  apiKey: process.env.OPENAI_KEY,
});

const response = await adapter.chat([
  { role: 'system', content: 'You are a helpful assistant.' },
  { role: 'user', content: 'What is 2+2?' },
]);

console.log(response.content); // "2 plus 2 equals 4"
```

### Example 2: Streaming with Tools

```typescript
import { AIAdapter } from 'primo-agent';
import { calculatorTool } from 'primo-agent';

const adapter = new AIAdapter({
  model: 'gpt-4',
  apiKey: process.env.OPENAI_KEY,
  useTools: true,
});

adapter.setTools([calculatorTool]);

const stream = adapter.chatStream([
  { role: 'user', content: 'Calculate 10 * 20' },
]);

let fullText = '';
for await (const event of stream) {
  if (event.type === 'text') {
    fullText += event.content;
    process.stdout.write(event.content);
  } else if (event.type === 'tool_call_start') {
    console.log(`\n[Calling tool: ${event.name}]`);
  } else if (event.type === 'tool_call_end') {
    console.log(`[Tool result: ${event.result}]`);
  }
}
```

### Example 3: Custom Compatible API

```typescript
// Using OpenRouter, Ollama, or other OpenAI-compatible endpoints
const adapter = new AIAdapter({
  model: 'anthropic/claude-3-sonnet',
  baseURL: 'https://openrouter.ai/api/v1',
  apiKey: process.env.OPENROUTER_KEY,
});
```
